Recommended Minimum

below are the essential elements for any web document

<meta charset ="utf-8"> <meta http-equiv="x-ua-compatible" content="ie-edge"> <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no"> <title>Page Title</title>

The above 3 meta tags *must* com first in the head to consistently ensure proper document rendering. Any ohter head element should come ager the tag.
